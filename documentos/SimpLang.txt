## Especificaciones del lenguaje SimpLang ##
- Tipos de datos soportados: `int`, `float`, `string`, `bool`
- Arrays: int[] numeros = [1,2,3,4,5]
- Tuplas: tuple persona = ("Juan", 30, True)
- Constantes: const int MAX_EDAD = 100
- Operadores soportados: `+`, `-`, `*`, `/`, `==`, `!=`, `<`, `>`, `<=`, `>=`
- Operador módulo: int resto = 10 % 3 #resto sera 1
- Operadores lógicos: and, or, not, xor (solo utilizados en condicionales o bucles)
- Estructuras de control: `if`, `elif`, `else`, `for`, `while`
- Funciones: definidas con `def`, con parámetros tipados
- Clases: con constructores, métodos y atributos, similar a Python pero con tipado estático como en Java

Estructura de programa principal
    def void main():
        print("Programa iniciado")
        # resto del código

Entrada y salida desde archivos
    file entrada = open("datos.txt", "r")
    string contenido = entrada.read()
    entrada.close()

Selección múltiple (switch-case)
    switch opcion:
        case 1:
            print("Opción 1 seleccionada")
            break
        case 2:
            print("Opción 2 seleccionada")
            break
        default:
            print("Opción no válida")

Bucle 'repeat until'
    repeat:
        print("Se ejecuta al menos una vez")
        edad += 1
    until edad == 18
    
Comentarios (Al encontrar # lo que siga hasta el \n sera un comentario)
    # Esto es un comentario

Variables (Deben declararse con su tipo)
    int edad = 25
    string nombre = "Juan" (un string debe ir entre "")
    bool activo = TRUE (siempre en mayúsculas)
    float salario = 3000.50 (siempre con un punto)

Funciones (Se definen usando def y se especifica el tipo de retorno)
    def int sumar(int a, int b):
        return a + b

Condicionales (if/elif/else)
    if edad > 18:
        print("Eres mayor de edad")
    elif edad == 18:
        print("Tienes 18 años")
    else:
        print("Eres menor de edad")

Bucle (for)
    for int i in range(0, 10):
        print(i)

Bucle (while)
    bool activo = TRUE
    while activo:
        print("El sistema está activo")
        activo = FALSE 


Clases y objetos (Soporte para constructores y métodos)
    class Persona:
        string nombre
        int edad

        def Persona(string nombre, int edad):
            self.nombre = nombre
            self.edad = edad

        def void saludar():
            print("Hola, mi nombre es " + self.nombre)

Entrada y salida (Al hacer print no hace falta especificar el tipo de variable)
    nombre = input("Introduce tu nombre: ")
    print("Hola, " + nombre)
